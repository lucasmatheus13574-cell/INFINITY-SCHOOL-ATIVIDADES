Variáveis CSS e Responsividade

Exemplo de layout responsivo usando CSS Custom Properties (Variáveis CSS)

Por que usar Variáveis CSS?

As variáveis CSS (também conhecidas como CSS Custom Properties) permitem armazenar valores reutilizáveis que podem ser facilmente atualizados em diferentes pontos do código. Isso é extremamente útil para criar layouts responsivos e manter consistência no design.

Benefícios:

  • Manutenção facilitada - altere uma variável e aplica em todo o site
  • Consistência de design - mesmo tamanho em múltiplos elementos
  • Responsividade simplificada - ajuste com @media queries
  • Redução de código repetitivo

Variáveis Definidas

Este documento usa variáveis CSS que se adaptam a diferentes tamanhos de tela:

📱 Mobile (até 768px)

Fonte H1: 24px
Espaçamento: 8px a 24px
Layout: 1 coluna

💻 Tablet (768px - 1023px)

Fonte H1: 32px
Espaçamento: 12px a 32px
Layout: 2 colunas

🖥️ Desktop (1024px+)

Fonte H1: 40px
Espaçamento: 16px a 40px
Layout: 3 colunas

Como Funciona

1. Definição das Variáveis (no :root)

As variáveis são definidas no seletor :root, o que as torna globais e acessíveis em todo o documento CSS. Cada variável começa com dois hífens (--).

2. Aplicação das Variáveis

Para usar uma variável, utilize a função var(--nome-da-variavel) em qualquer propriedade CSS.

3. Modificação em Media Queries

Dentro de @media queries, você pode redefini as variáveis do :root para que todos os elementos usando essas variáveis sejam atualizados automaticamente.

Exemplo de Código

Definição no :root:
:root {
  --font-size-h1: 24px;
  --spacing-large: 24px;
}

Aplicação em elementos:
h1 {
  font-size: var(--font-size-h1);
  margin-bottom: var(--spacing-large);
}

Modificação em Media Query:
@media (min-width: 768px) {
  :root {
    --font-size-h1: 32px;
    --spacing-large: 32px;
  }
}

Dicas Importantes

✓ Use :root

Defina variáveis globais no seletor :root para acessibilidade em todo o documento.

✓ Nomenclatura Clara

Use nomes descritivos como --spacing-large, --font-size-h1 para melhor compreensão.

✓ Teste em Vários Tamanhos

Verifique seu layout em mobile, tablet e desktop para garantir responsividade.
